# 画像フォーマット比較デモ

ラスター画像とベクター画像の違いを視覚的に比較するためのデモプロジェクトです。

## 目的

このプロジェクトは、以下の目的で作成されています：

1. **ラスター画像とベクター画像の違いを理解する**
   - ラスター画像: ピクセルの集合で構成され、写真や複雑なグラデーションに適している
   - ベクター画像: 数式で定義され、拡大しても劣化しない

2. **各画像フォーマットの特性を比較する**
   - PNG: 可逆圧縮、透過対応、高品質
   - JPEG: 非可逆圧縮、写真に最適、小サイズ
   - WebP: 次世代フォーマット、高圧縮率
   - GIF: アニメーション対応、256色制限
   - SVG: ベクター形式、無限に拡大可能

3. **ファイルサイズの違いを確認する**
   - 各フォーマットのファイルサイズを実際に計測・表示
   - 用途に応じた最適なフォーマット選択の参考にする

4. **拡大時の品質劣化を視覚的に確認する**
   - ラスター画像は拡大すると品質が低下
   - ベクター画像は拡大しても常に鮮明

## プロジェクト構成

```
image-examples/
├── README.md              # このファイル
├── package.json           # プロジェクト設定と依存関係
├── scripts/
│   └── generate-images.js # 画像生成スクリプト
└── public/
    ├── index.html         # デモページ
    └── assets/
        ├── css/           # スタイルシート
        └── img/           # 画像ファイル
            ├── sample.png
            ├── sample.jpg
            ├── sample.webp
            ├── sample.gif
            └── sample.svg
```

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 画像の生成

```bash
npm run generate
```

このコマンドで `public/assets/img/` ディレクトリに各フォーマットのサンプル画像が生成されます。

### 3. デモページの起動

```bash
npm run serve
```

ブラウザが自動的に開き、デモページ（http://localhost:8080）が表示されます。

## 使い方

1. ブラウザでデモページを開くと、各画像フォーマットが並んで表示されます
2. 各画像カードには以下の情報が表示されます：
   - フォーマット名
   - ファイルサイズ（自動計算）
   - 圧縮方式
   - 透過対応の有無
   - 特徴
3. ページ下部の「拡大時の品質比較」セクションで、ラスター画像とベクター画像の拡大時の違いを確認できます

## 技術スタック

- **HTML/CSS/JavaScript**: フロントエンド
- **Node.js**: 画像生成スクリプト
- **Canvas API**: ラスター画像の生成（node-canvas）
- **SVG**: ベクター画像の定義

## 各フォーマットの使い分け

### ラスター画像

- **PNG**: ロゴ、アイコン、透過が必要な画像
- **JPEG**: 写真、複雑な色の画像
- **WebP**: Web向けの画像全般（モダンブラウザ）
- **GIF**: シンプルなアニメーション

### ベクター画像

- **SVG**: ロゴ、アイコン、図形、グラフ、無限に拡大する必要がある画像
